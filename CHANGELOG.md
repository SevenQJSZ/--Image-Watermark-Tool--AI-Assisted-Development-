# Changelog

本项目的所有重要变更将记录在此文件中。

格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.1.0] - 2026-07-23

### 新增
- PNG 图片水印模式：可选择透明底 PNG 作为水印，支持大小比例、不透明度、旋转角度、行间距、列间距、列偏移调节
- 文字 / 图片水印类型一键切换，两套参数独立保存
- exe 启动时自动检测 Edge WebView2 运行时，缺失则弹窗引导至微软官方下载页

### 变更
- 产品名称由"图片文字水印工具"调整为"图片水印工具"
- README 新增"本软件完全免费，任何需要付费的都是骗子"防诈骗郑重声明

## [1.0.0] - 2026-07-23

### 新增
- 首个发布版本
- 文字水印：自定义文字、字体、颜色、字号、不透明度、旋转角度
- 平铺布局参数：行间距、列间距、列偏移（奇数行梅花桩错位）
- 多图批量选择、缩略图切换、实时预览
- 导出保持原图像素尺寸，支持保持原格式 / PNG 无损 / JPEG 自定义质量
- PyInstaller 单文件 exe 打包，pywebview 桌面窗口 + 原生保存对话框
